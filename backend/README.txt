# Backend - Smart Rail QR Park

Spring Boot REST API for the Smart Rail QR Park project.

## Tech Stack
- Java 17+
- Spring Boot 3.2.0
- Maven 3.6+
- PostgreSQL (Supabase)

## Setup

1. Install Java 17+ and Maven 3.6+
2. Configure database in `src/main/resources/application.properties`
3. Run: `mvn clean install`
4. Start: `mvn spring-boot:run`

Server runs at: http://localhost:3020

## API Endpoints
- GET /api/welcome - Welcome message
